# Spring Boot Sample

Minimal Spring Boot 4 app for Git practice.

## Requirements
- Java 21
- Maven wrapper included

## Run
```bash
./mvnw spring-boot:run
```

Windows:
```powershell
mvnw.cmd spring-boot:run
```

## Test
```bash
./mvnw test
```

## Notes
- Entry point: `src/main/java/com/example/demo/DemoApplication.java`.
- No controllers are defined yet; add one to expose endpoints.
